# Do / Don’t — TDAH.blog

## Do
- usar coluna estreita (62ch)
- usar serif moderna no conteúdo
- usar contraste médio e fundo tipo papel
- usar TOC em texto longo
- usar callouts para pausas cognitivas
- usar espaços generosos

## Don’t
- branco puro (#ffffff)
- preto puro (#000000)
- roxo saturado
- layout largo
- grids múltiplos em leitura
- fontes finas
- animações decorativas
- estética SaaS
- visual jovem/adolescente
